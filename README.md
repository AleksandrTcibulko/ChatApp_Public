### ChatApp

Stack:
- Programmatic UI (w/o using Interface Builder)
- Compositional Layout and Diffable DataSource
- Firebase Auth with Email
- Sign In with Google Account
- Firebase Firestore
- Firebase Storage
- MessageKit (sending text messages and pictures)
- SDWebImage
